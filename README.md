# IT Гараж

Эта GitHub-организация закрытого сообщества с обучением IT Гараж, доступ только у участников сообщества.

Здесь собраны все необходимые материалы из обучения: схемы, промпты, mcp, скиллы и так далее. Материалы обновляются по мере выхода новых видео.

## Доступ и использование
Как платный участник, вам предоставляются права на использование в личных и коммерческих проектах, но распространение за пределами сообщества запрещено.
