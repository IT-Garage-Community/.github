# IT Гараж

Эта GitHub-организация закрытого сообщества с обучением "IT Гараж".

Здесь собраны все необходимые материалы для обучения и использования в своих проектах: схемы, инструкции, промпты, mcp, скиллы и так далее. Материалы постоянно обновляются.

## Доступ и использование

Доступ предоставляется только участникам сообщества, инструкция находится в закрытом канале.

⚠️ Все репозитории организации регулируются [Лицензией](https://github.com/IT-Garage-Community/.github/blob/main/LICENSE.md). Пожалуйста, ознакомься с условиями перед использованием материалов.

## Навигация

1. [Общее](https://github.com/IT-Garage-Community/common) – материалы, не связанные с конкретной веткой обучения.
2. [Вайбкодинг](https://github.com/IT-Garage-Community/vibecoding) – материалы ветки по вайбкодингу.
