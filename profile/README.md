# IT Гараж

Эта GitHub-организация закрытого сообщества с обучением "IT Гараж".

Здесь собраны все необходимые материалы для обучения: схемы, инструкции, промпты, mcp, скиллы и так далее. Материалы постоянно обновляются.
## Доступ и использование
Доступ предоставляется только участникам сообщества, инструкция находится в закрытом канале.

⚠️ Участники могут использовать материалы в личных и коммерческих проектах, но распространение за пределами сообщества запрещено.
